# elasticsearch 실습 세팅 (single node)

### docker build
```shell
docker build -t fastcampus-es-elasticsearch . 
```

### docker run
```shell
docker-compose up -d
```

### docker stop
```shell
docker-compose down
```

### elasticsearch 접속 url
* http://localhost:9200

### kibana 접속 url
* http://localhost:5601
